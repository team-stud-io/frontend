# Stud.io 데모데이

Stud.io는 시험 일정, 과목별 학습 범위와 현재 학습 상태를 바탕으로 AI 학습 전략과 일일 학습 계획을 제공하는 학습 관리 서비스입니다.

## 배포

- 웹 데모: [https://studio-six-tawny-65.vercel.app/auth/login](https://studio-six-tawny-65.vercel.app/auth/login)
- 배포 플랫폼: Vercel

## 주요 데모 흐름

1. 로그인 화면에서 서비스를 시작합니다.
2. AI 튜터에서 시험 유형·일정·과목·학습 범위·목표 성적·학습 스타일·가능 시간을 입력합니다.
3. AI 전략 생성 작업의 진행 상태를 확인하고, 과목별 전략과 주차별 학습 계획을 조회합니다.
4. 홈에서 오늘의 학습 할 일을 확인·완료 처리하고, 할 일별 학습 팁과 일일 회고를 기록합니다.
5. 전략을 저장하거나 플래너에 적용해 이후 학습 계획에 반영합니다.

## 구현 특징

- Expo Router의 파일 기반 라우팅과 타입 경로를 사용합니다.
- Android·iOS·웹에서 동일한 화면 구조를 제공하고, 안전 영역과 Android edge-to-edge 화면을 고려합니다.
- AI 전략 생성 중 앱을 다시 열어도 진행 중인 생성 작업과 튜터 세션을 복구합니다.
- 백엔드의 snake_case API 계약은 공통 클라이언트에서 처리하고, 프론트엔드 코드는 일관된 camelCase 모델을 사용합니다.
- 실제 환경별 API 주소와 민감한 설정 값은 저장소에 포함하지 않습니다.
